# Mini Projeto EduConnect 🎓

Este é um miniprojeto simples de cadastro de alunos desenvolvido para a atividade prática de Git e GitHub da disciplina de Desenvolvimento Web.

## 🚀 Funcionalidades
- Cadastro de nome do aluno.
- Cadastro de e-mail do aluno (implementado via branch de feature).
- Listagem dinâmica dos alunos cadastrados em tela.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (ES6)
- Git e GitHub para versionamento
